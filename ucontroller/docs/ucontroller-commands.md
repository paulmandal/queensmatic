# uController Command Definitions

## Configuration Update

Format: C:n

| Placeholder | Meaning                                       |
| ------------| ----------------------------------------------|
| n           | New number of LEDs the uController is driving |

## LED Update

Format: U:n,r,g,b,br

| Placeholder | Meaning             |
| ------------| --------------------|
| n           | LED # to update     |
| r           | Red channel value   |
| g           | Green channel value |
| b           | Blue channel value  |
| br          | Brightness          |
